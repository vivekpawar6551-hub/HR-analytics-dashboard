# HR-analytics-dashboard

Overview
This project is an interactive HR Analytics Dashboard built using Power BI. The dashboard helps organizations analyze employee data, monitor attrition trends, and gain insights into workforce performance through dynamic visualizations and KPI tracking.

Features
Employee Attrition Analysis
Department-wise Employee Distribution
Job Satisfaction Insights
Age Group Analysis
Education Field Analysis
Gender Distribution
Interactive Filters & Slicers
KPI Cards for Key Metrics
KPIs Included
👤 Average Age
👥 Active Employees
📉 Attrition Rate
🚪 Attrition Count
🏢 Total Employees
Tools & Technologies Used
Power BI
Power Query
DAX
Data Modeling
Data Visualization
Dashboard Insights
Analyze employee attrition patterns
Identify departments with high attrition
Understand workforce demographics
Track employee distribution by role and education
Improve HR decision-making using data insights
Skills Demonstrated
Dashboard Design
Data Cleaning & Transformation
DAX Calculations
Business Intelligence
Data Analysis
Storytelling with Data
Project Objective

The main objective of this project is to transform raw HR data into meaningful insights that help businesses make better workforce and employee retention decisions.
